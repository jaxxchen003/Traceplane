# Claude Code + Traceplane 集成示例

本示例演示如何将 Claude Code Agent 接入 Traceplane 平台。

## 快速开始

### 1. 环境配置

在项目根目录创建 `.env` 文件：

```bash
TRACEPLANE_BASE_URL=http://localhost:3000
TRACEPLANE_PROJECT_ID=your-project-id
TRACEPLANE_AGENT_ID=claude-code
```

### 2. 创建 MCP 配置

```bash
npm run mcp:host -- claude-code .mcp.json
```

### 3. 在 Claude Code 中使用

Claude Code 现在可以通过 MCP 工具与 Traceplane 交互：

- `create_episode` - 创建新的 Episode
- `append_trace` - 记录执行步骤
- `write_memory` - 创建 Memory 条目
- `create_artifact` - 生成产物

### 4. 自动上报示例

在 `.claude/hooks/` 目录创建 `traceplane.ts`：

```typescript
import { TraceplaneSDK } from '@traceplane/agent-sdk';

const sdk = new TraceplaneSDK({
  baseUrl: process.env.TRACEPLANE_BASE_URL,
  projectId: process.env.TRACEPLANE_PROJECT_ID,
  agentId: 'claude-code'
});

export async function onEpisodeStart(title: string) {
  return await sdk.startEpisode({
    title,
    goal: 'Generated by Claude Code',
    successCriteria: 'Task completed'
  });
}

export async function onStepComplete(step: string, result: string) {
  const session = sdk.getCurrentSession();
  await session?.step(step, result);
}
```

### 5. 验证集成

运行测试：

```bash
npm run claude:hook:test
```

## 使用 Traceplane SDK

### 直接集成

```typescript
import { TraceplaneSDK } from '@traceplane/agent-sdk';

class ClaudeTraceplaneReporter {
  private sdk: TraceplaneSDK;
  private session: any;

  constructor() {
    this.sdk = new TraceplaneSDK({
      baseUrl: process.env.TRACEPLANE_BASE_URL!,
      projectId: process.env.TRACEPLANE_PROJECT_ID!,
      agentId: 'claude-code',
      debug: true
    });
  }

  async startTask(title: string, goal: string) {
    this.session = await this.sdk.startEpisode({
      title,
      goal,
      successCriteria: 'Task completed successfully'
    });
    console.log(`Episode started: ${this.session.getCurrentEpisode()?.id}`);
  }

  async recordStep(step: string, result: string) {
    await this.session?.step(step, result);
  }

  async recordToolUse(tool: string, input: string, output: string) {
    await this.session?.toolUse(tool, input, output);
  }

  async completeTask(outcome: string) {
    await this.session?.complete(outcome);
    console.log('Episode completed');
  }
}

export const reporter = new ClaudeTraceplaneReporter();
```

### 在 Claude Code 中使用

```typescript
import { reporter } from './traceplane-reporter';

async function analyzeCode(files: string[]) {
  await reporter.startTask(
    'Code Analysis',
    'Analyze codebase for security issues'
  );

  await reporter.recordStep(
    'Scan files',
    `Scanned ${files.length} files`
  );

  await reporter.recordToolUse(
    'security-scanner',
    'scan --severity high',
    'Found 3 issues'
  );

  await reporter.completeTask('Analysis complete');
}
```

## 查看结果

运行后打开 Traceplane UI：

```
http://localhost:3000/zh/projects/{projectId}
```

你将看到：
- 新的 Episode 出现在列表中
- 执行时间线记录了每个步骤
- Memory 中保存了关键发现
- Artifact 中存储了分析报告
- Event Stream 实时显示事件

## 进阶配置

### 自动错误捕获

```typescript
const session = await sdk.startEpisode({...}, {
  autoCaptureErrors: true  // 自动捕获未处理异常
});
```

### 实时事件订阅

```typescript
const client = new TraceplaneClient(config);

const unsubscribe = client.subscribeEvents(
  episodeId,
  (event) => {
    console.log('New event:', event);
  }
);
```

### 批量上报

```typescript
const session = await sdk.startEpisode({...}, {
  autoCreateTraces: false,  // 禁用自动上报
  autoFlushInterval: 5000    // 每 5 秒批量上报
});

// 手动批量上报
await session.flush();
```
