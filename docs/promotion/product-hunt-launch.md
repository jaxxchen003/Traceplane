## 短描述（140字内）
Traceplane 让多个 AI Agent（Claude Code、OpenCode、Gemini CLI）的工作接成一条连续、可回溯、可交接的主线。不是日志，是 Agent 的意识载体。

## 详细描述
当你在同时使用多个 AI Agent 协作时，是否遇到过：
- 同样的上下文要反复讲给不同 Agent
- Agent 做到一半，另一个接不上
- 工作散落在聊天记录里无法复盘

Traceplane 解决的不是让 Agent 更聪明，而是让 Agent 之间的工作**不断档**。

### 核心功能
- **Episode 连续性模型**：Agent 的意识载体，可恢复、可复用、可交接
- **Surgical Replay**：像 git checkout 一样回到 Agent 工作的任意状态
- **Handoff Brief**：自动生成无损交接文档
- **全链路观测**：Agent 行为、Artifact、风险指标一体化追踪

### 技术栈
- Next.js 16 + React 19 + Tailwind v4
- SQLite（本地）/ Supabase Postgres（云端）
- MCP (Model Context Protocol) 通用接入
